##如何在项目中使用Putils  
----------------------------
Gradle:
```
    compile 'com.jph.putils:putils_library:1.0.0'
```

Maven:
```
<dependency>
  <groupId>com.jph.putils</groupId>
  <artifactId>putils_library</artifactId>
  <version>1.0.0</version>
  <type>pom</type>
</dependency>
```
